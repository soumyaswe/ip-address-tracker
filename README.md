A Streamlit web app to find the geolocation of any IP address. Enter an IP address to view its city, state, country, latitude, longitude, and ISP information. The app also displays your current public IP address.

## Features

- Get geolocation details for any IP address
- View your current public IP
- Stylish, user-friendly interface
- Team information in the sidebar

## Installation

1. Clone this repository or download the source code.
2. Install dependencies using pip:

   ```sh
   pip install -r requirements.txt
   ```

## Usage
Run the Streamlit app:
streamlit run [app.py](app.py)
